# SobatKost - Digital Rental Management System

**SobatKost** adalah aplikasi mobile berbasis Flutter yang dirancang untuk mendigitalisasi pengelolaan kamar kos. Aplikasi ini memungkinkan penghuni untuk mengelola data penyewa, melaporkan kerusakan (maintenance), dan menerima notifikasi tagihan atau informasi penting secara real-time.

Proyek ini disusun untuk memenuhi kriteria **Mini Project (ETS) Mata Kuliah Pemrograman Perangkat Bergerak**.

## 🎯 Kriteria Penilaian (Checklist)

Aplikasi ini telah mengimplementasikan seluruh poin persyaratan tugas:

- [x] **CRUD dengan Relational Database (10%)**: Menggunakan `sqflite` untuk manajemen data lokal (Contoh: Daftar Inventaris Kamar/Catatan Pengeluaran).
- [x] **Firebase Authentication (5%)**: Fitur Register dan Login untuk keamanan akses pengguna.
- [x] **Storing Data in Firebase (5%)**: Sinkronisasi data utama (Contoh: Tiket Maintenance/Profil) ke Cloud Firestore.
- [x] **Notifications (5%)**: Implementasi notifikasi lokal sebagai pengingat atau konfirmasi aksi sukses.
- [x] **Smartphone Resource (5%)**: Integrasi fitur **Kamera** untuk mengunggah bukti fisik atau foto kondisi fasilitas.
- [x] **Demo Video & GitHub (10%)**: Dokumentasi lengkap berupa kode sumber di GitHub dan video demonstrasi fitur.

## 🛠️ Tech Stack & Library

- **Framework:** Flutter
- **Language:** Dart
- **Database Lokal:** [sqflite](https://pub.dev/packages/sqflite)
- **Backend:** Firebase (Auth & Firestore)
- **Hardware Integration:** [image_picker](https://pub.dev/packages/image_picker) (Camera)
- **Local Notification:** [flutter_local_notifications](https://pub.dev/packages/flutter_local_notifications)
